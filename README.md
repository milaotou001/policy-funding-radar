# 政策资金流向雷达

<div align="center">

**中国十五五规划 × 9城市落地证据 × 市场资金验证 —— 三层交叉判断的产业政策投资研究工具**

</div>

---

## 项目概述

本工具将中国政府的产业政策信号（从十四五到十五五规划纲要的变化）与省级落地证据（浙江省 + 9 个重点城市）和市场资金流向（ETF 行情数据）进行**三层交叉验证**，输出 34 个产业的**可证伪方向判断**。

> **核心逻辑**：政策优先级是必要条件，不是充分条件——你要确认钱真的在往下拨（省级/城市落地证据），资金真的在往里走（市场验证），才构成一个可操作的信号。

### 与常见政策分析的区别

| | 常见做法 | 本工具 |
|---|---|---|
| 信号来源 | 单一文件解读 | **五层叠合**：国家规划 + 省级落地 + 城市报告 + 城市规划 + 市场资金 |
| 判断方式 | 主观定性 | **三维交叉矩阵**：政策动量 × 执行证据 × 市场认知 |
| 覆盖范围 | 泛泛而谈 | **34 个产业**，每个产业逐层打分 |
| 可证伪性 | 无 | **每个判断绑定证伪条件**——什么情况出现说明你错了 |
| 样本局限 | 不披露 | **明确标注"样本偏差"**——承认哪些产业我们看不准 |

## 方向判断系统

不再数绿点。三个维度交叉判断：

```
政策动量（加速/稳定/减速/初现）
    ×
执行证据（强执行/有执行/弱执行/无证据/样本偏差）
    ×
市场认知（资金认同/观望/分歧/流出/无数据）
    ↓
信念等级（高信念 / 中等信念 / 低信念 / 观望 / 回避）
    + 矛盾标记（如"政策强推但市场未认"）
    + 证伪条件（如"省级降级→减仓"）
```

### 🟢 矛盾检测

系统自动识别三种跨层矛盾：

- **政策强推但市场未认** → 可能是左侧机会，也可能政策空转
- **政策退潮但资金未撤** → 注意见顶风险
- **资金先行但无落地证据** → 警惕炒作退潮

### 🔴 证伪条件

每个信念等级绑定具体的可观测证伪条件，并在界面中显示**当前监控状态**（红黄绿灯），告诉你离证伪触发还有多远。

## 数据覆盖

### 五层信号

| 层 | 来源 | 覆盖范围 | 更新频率 |
|---|---|---|---|
| 国家规划 | 十四五/十五五规划纲要 | 全国 | 每 5 年 |
| 省级落地 | 浙江省十五五规划 | 浙江 1 省 | 每 5 年 |
| 工作报告 | 国务院/浙江省/9 市政府工作报告 | 全国+浙江+9 城 | **每年 1-3 月** |
| 城市规划 | 9 市十五五规划纲要 | 9 城 | 每 5 年 |
| 市场验证 | 公开 ETF 行情数据 | 32/34 产业 | **每周可刷新** |

### 9 城样本

上海 · 深圳 · 杭州 · 南京 · 苏州 · 北京 · 广州 · 合肥 · 武汉

**样本局限**：9 城全部偏向东部沿海和科技中心。深海、核能、关键矿产 3 个产业标记为"样本偏差"——它们的产业重心不在这些城市，执行证据需另行验证。

### 34 个产业（按文件优先级分层）

| 层级 | 产业 |
|---|---|
| **攻坚战** | 半导体/芯片、工业母机、先进材料、基础软件/工业软件、生物制造、人工智能 |
| **增长引擎** | 量子科技、机器人/具身智能、脑机接口、6G、氢能/核聚变 |
| **能源与资源安全** | 新能源、核能、新型储能、关键矿产 |
| **战略产业** | 低空经济、数字经济、商业航天、生物医药、智能驾驶 |
| **强化升级** | 电力设备、医疗器械、国产大飞机、新能源车、深海经济、数据要素、农业科技、中医药、种业 |
| **制度民生/监管化** | 绿色金融、银发经济、冰雪经济、碳市场、平台经济 |

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/milaotou-tools/policy-funding-radar.git
cd policy-funding-radar

# 安装依赖
npm install

# 拉取最新市场数据（每周运行一次）
node scripts/fetch-market-data.cjs
node scripts/merge-market.cjs

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

在线版本：**https://milaotou-tools.github.io/policy-funding-radar/**

| 窗口 | 内容 | 来源 |

## 免责声明

**本工具仅供研究参考，不构成任何形式的投资建议。** 产业标签归类和高亮标注为 AI 语义分析结果，可能存在偏差，请以原始政策文件为准。ETF 映射使用市场上最接近的标的，部分新兴产业尚无纯正 ETF 上市，使用代理 ETF 的涨跌幅不能直接等同于该产业的市场表现。过往政策信号强度与未来投资收益之间不存在确定的因果关系。
