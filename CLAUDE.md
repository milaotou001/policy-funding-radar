# CLAUDE.md

## 设计系统
生成任何 UI 代码前，必须先读取项目根目录的 `DESIGN.md`。
只使用其中定义的颜色、字体、间距。违反 DESIGN.md 约束的代码视为不合格。

## 项目概述
政策资金流向雷达 — 中国产业政策→投资信号分析工具。
React + Vite + TypeScript + Tailwind CSS v4。34 个产业，静态 JSON 数据。

## 核心规则
- 数据区（表格、矩阵）追求紧凑，文本区（证据阅读）追求舒适行距
- 二级信息默认折叠（`<details>`）
- 所有数值列 monospace 右对齐
- 不用阴影、渐变色、动画
- 代理 ETF 数据必须标注 `*`
